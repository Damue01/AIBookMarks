功能亮点

🤖

AI 智能分类

支持 OpenAI / Claude / Ollama / 自定义 API，AI 分析书签内容并自动建议分类方案。

📂

分类规划

AI 先生成目录结构方案，你确认后再执行，全程可控，拒绝混乱。

⚡

新书签自动归类

每次收藏新书签时实时分析，页面内 Toast 通知提示归类建议。

📏

自定义规则

支持域名匹配、通配符、正则表达式，规则优先于 AI，精准控制。

🔄

拖放排序

拖动书签和文件夹调整顺序，拖入文件夹快速移动，操作流畅。

💾

备份与恢复

整理前自动备份，支持导入/导出 JSON，随时一键回滚，安全无忧。

🌐

中英双语

界面完整支持中文和 English，无缝切换。

🔒

隐私安全

用户自持 API Key，数据本地化处理，开源代码透明可查。
